== Description == 

Adds a sidebar widget to display Twitter updates (uses the Javascript <a href="http://twitter.com/badges/which_badge">Twitter 'badge'</a>)

== Installation == 

Copy the 'twitter-widget.php' file to your WordPress '/wp-content/plugins/widgets' folder.
